# iOS 앱 테스트 및 기기 설치 가이드 (Capacitor)

이 문서는 Dimmer Switch 앱(Next.js + Capacitor)을 Mac의 iOS 시뮬레이터에서 테스트하는 방법과, 실제 사용자님의 iPhone 기기에 직접 설치하는 방법을 안내합니다.

## 📱 파트 1: Mac 시뮬레이터에서 테스트하기 (케이블 불필요)
케이블이 없거나 빠르게 개발을 진행해야 할 때 가장 추천하는 방법입니다. 로컬 푸시 알림도 실제 폰과 100% 동일하게 동작합니다.

1. **최신 코드 빌드 및 동기화**
   터미널에서 앱의 최신 변경사항을 iOS 프로젝트에 반영합니다.
   ```bash
   npm run build && npx cap sync
   ```

2. **Xcode 실행**
   터미널에서 아래 명령어를 입력하여 Xcode를 엽니다.
   ```bash
   npx cap open ios
   ```

3. **시뮬레이터 선택**
   - Xcode가 열리면 화면의 맨 위 중앙에 있는 기기 선택창(기본값: `Any iOS Device`)을 클릭합니다.
   - 드롭다운 목록에서 **iPhone 16** (또는 원하는 기기)를 선택합니다.

4. **실행 및 푸시 테스트**
   - 왼쪽 상단의 **▶ (Play)** 버튼을 누르거나 단축키 `Cmd + R`을 누릅니다.
   - 잠시 후 가상의 아이폰 창이 뜨고 앱이 실행됩니다.
   - 앱 내에서 [알림 활성화] 및 [백그라운드 푸시] 스위치를 켭니다.
   - **`Cmd + Shift + H`** 를 눌러 앱을 배경으로 내리거나 종료한 뒤, 정해진 시간이나 테스트 트리거에 따라 알림이 화면 상단에 잘 도착하는지 확인합니다.

---

## 📲 파트 2: 실제 내 iPhone에 설치하기 (케이블 필수)
앱을 본인의 실제 아이폰에 넣어 일상생활에서 사용해보고 싶을 때 진행하는 방법입니다. 최초 1회는 반드시 **USB 케이블 연결**이 필요합니다.

### 사전 준비: Apple ID 로그인
1. Xcode 상단 메뉴에서 **Xcode > Settings** (또는 Preferences)를 클릭합니다.
2. **Accounts** 탭으로 이동하여 왼쪽 아래의 **+** 버튼을 누릅니다.
3. **Apple ID**를 선택하고 본인의 애플 계정으로 로그인합니다. (무료 계정도 가능)

### 기기 연결 및 인증서 설정
1. **아이폰 연결**: Mac과 iPhone을 USB 케이블로 연결합니다. 아이폰 화면에 "이 컴퓨터를 신뢰하겠습니까?" 창이 뜨면 **[신뢰]**를 누르고 비밀번호를 입력합니다.
2. **개발자 모드 활성화**: 아이폰에서 **[설정] > [개인정보 보호 및 보안] > [개발자 모드]**를 켭니다. (스마트폰이 재부팅될 수 있습니다)
3. **Xcode 타겟 선택**: Xcode 상단 중앙의 기기 선택창을 눌러 목록 상단에 있는 **본인의 iPhone 이름**을 선택합니다.
4. **Signing (서명) 설정**:
   - Xcode 왼쪽 파일 탐색기(네비게이터)에서 맨 위에 있는 파란색 아이콘 **App** 프로젝트를 클릭합니다.
   - 메인 화면 중간의 **Signing & Capabilities** 탭을 클릭합니다.
   - **Automatically manage signing** 체크박스에 체크합니다.
   - **Team** 드롭다운에서 방금 로그인한 본인의 이름(Personal Team)을 선택합니다.
   - *Bundle Identifier*가 고유한지 확인합니다. (예: `com.mannlab.dimmerswitch`)

### 설치 및 폰에서 신뢰 허용
1. 왼쪽 상단의 **▶ (Play)** 버튼(`Cmd + R`)을 누릅니다.
2. 빌드가 완료되면 아이폰에 앱 아이콘이 설치됩니다.
3. 앱을 실행하려고 할 때 "신뢰할 수 없는 개발자" 경고가 뜬다면:
   - 아이폰 **[설정] > [일반] > [VPN 및 기기 관리]** 로 이동합니다.
   - '개발자 앱' 항목 아래에 있는 본인의 Apple ID를 누르고 **[신뢰]**를 클릭합니다.
4. 이제 홈 화면에서 Dimmer Switch 앱을 자유롭게 실행하고 사용할 수 있습니다!
